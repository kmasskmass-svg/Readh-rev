(ج) الوصول لمصادر التمويل الحكومي.                     (د) توظيف التقنية الحديثة كالحوسبة السحابية.

32. العملية التي يتعرف من خلالها صاحب الفكرة على مدى جدواها قبل البدء الفعلي، وتساعده على اكتشاف نقص المعلومات وتقدير العائد، تُسمى:
(أ) بناء شبكة العلاقات الشخصية.                             *(ب) إجراء الدراسة الأولية للمشروع.

(أ) الإنتاج، التسويق، التمويل، تكنولوجيا المعلومات.                  *(ب) التخطيط، التنظيم، التوجيه، الرقابة.
(ج) الابتكار، الإبداع، المخاطرة، العصف الذهني.                      (د) البحث، التطوير، التصميم، التوزيع اللوجستي.

34. أي مما يلي يُعد من فوائد الارتباط بالإنترنت وتوظيف "التقنية الحديثة" بشكل فاعل في المشروعات الريادية الناشئة؟
(أ) زيادة تكاليف التشغيل الإداري بشكل مبالغ فيه.                     *(ب) أتمتة الأعمال الإدارية واستخدام التخزين السحابي لتنظيم العمل.

35. الثقافة المجتمعية التي تعتبر رافداً أساسياً وداعماً لنمو ريادة الأعمال هي الثقافة التي:
(أ) تدعم بشدة مفاهيم التقليد والرتابة والنمطية في الوظائف.                      (ب) تنبذ الفشل وتعاقب عليه وترفض مبدأ المخاطرة تماماً.

(ج) تقديم قروض مباشرة بدون فوائد للمنشآت متناهية الصغر.
(د) إصدار براءات الاختراع للمبتكرين السعوديين.

37. تعتبر القدرة على بناء "شبكة العلاقات" (Networking) من المهارات الضرورية للنجاح لأنها:
*(أ) تساعد في تبادل الأفكار والبحث عن شركاء وتمويل وترويج المشروع تجارياً.
(ب) تزيد من ساعات العمل الإداري اليومي بشكل غير مبرر.
(ج) تُغني رائد الأعمال كلياً عن إجراء دراسة أولية أو خطة عمل لمشروعه.
(د) تضمن تحقيق أرباح واستقرار الدخل بنسبة 100% دون جهد يذكر.

38. ما هي الأداة النفسية والسلوكية التي يجب على رائد الأعمال التسلح بها لمواجهة "الإحباط" المتوقع في بداية التأسيس؟
(أ) التوجه للانسحاب الفوري وإعلان الإفلاس عند أول خسارة مالية.         
*(ب) التمتع بالصبر والتفاؤل والإصرار لتجاوز التحديات واختراق العوائق.
(ج) تجنب استشارة المرشدين وتحمل المسؤولية بشكل منفرد بعيداً عن فريق العمل.
(د) الاعتماد على الحظ والتوقعات العشوائية وتجاهل بيانات السوق.

39. توفير شريحة واسعة من العملاء المحتملين الذين يتمتعون بالقوة الشرائية والمرونة في تبني المنتجات الجديدة يُعبر عن أحد عناصر المنظومة وهو:
(أ) السياسات الحكومية.   (ب) رأس المال البشري.    *(ج) السوق الواعدة.    (د) المؤسسات المالية.

40. شركة "عمل المستقبل" هي إحدى الجهات الداعمة والممكنة في المملكة التي تركز أهدافها ومبادراتها بشكل خاص على:
*(أ) تنشيط اقتصاد العمل الحر للمواطنين في المملكة.        (ب) تنظيم المنافسة العادلة ومكافحة الممارسات الاحتكارية.
(ج) جمع الزكاة والضرائب من الشركات الأجنبية.            (د) تقديم خدمات البريد والخدمات اللوجستية غير التقليدية (سبل).
 
الوحدة الثالثة: الشخصية الريادية

41. تُعرّف "العقلية الريادية" في أدبيات الأعمال المعاصرة بأنها:
(أ) طريقة التفكير التي تتميز بالخوف من التغيير والتمسك الصارم بالحلول التقليدية.
*(ب) طريقة التفكير التي تتميز بالإبداع والابتكار والمخاطرة والقدرة على تحويل الفرص لمشاريع.
(ج) القدرة الحصرية على إدارة المشروعات الحكومية بأقل التكاليف المالية الممكنة.
(د) المهارة الفنية لتشغيل الآلات والمعدات الميكانيكية بفعالية فقط.

42. أي من الخيارات التالية لا يُعد من الفوائد والمكتسبات المباشرة عند تبني العقلية الريادية؟
(أ) التغلب على التحديات بطرق مبتكرة وتخطي العقبات.
(ب) التفكير خارج الصندوق وحل المشكلات بطرق غير اعتيادية.
*(ج) تجنب المخاطر بشكل كلي والاعتماد الدائم على الآخرين في اتخاذ القرارات.
(د) إلهام الآخرين، وامتلاك شعور أقوى بالذات، وتحمل المسؤولية عن النتائج.

43. من السمات السبع الأساسية لرائد الأعمال سمة "المثابرة والإصرار"، والتي تعني بشكل دقيق:
(أ) القدرة اللفظية على إقناع الآخرين بوجهة نظره بقوة.                   *(ب) العمل المتواصل والصبر للوصول للهدف.
(ج) تقبل آراء الآخرين بصدر رحب والعمل على تنفيذها فوراً.             (د) البحث المستمر عن أحدث التقنيات والوسائل الإلكترونية.

44. السمة التي تجعل رائد الأعمال يمتلك طاقة إيجابية جياشة وطموحاً عالياً، ويستمتع بعملية إدارة مشروعه كما يستمتع الفنان بفنه تُسمى:
(أ) الحساسية للمشكلات.           *(ب) الحماس والشغف.             (ج) تقبل النقد.          (د) الطلاقة اللفظية.

45. يتميز رائد الأعمال الناجح بمرونة كبيرة لتقبل آراء الآخرين والاستفادة من نصائحهم ومشورتهم لتطوير مشروعه، هذه السمة تُعرف بـ:
(أ) الثقة العالية بالنفس.                 (ب) عدم خشية الفشل.             *(ج) يتقبل النقد.        (د) القدرة العالية على الإقناع.

46. المهارات المتعلقة بإتقان العمل في جزء معين من المشروع بصورة تخصصية، مثل (كيفية تصميم إعلان أو برمجة تطبيق إلكتروني)، تُصنف على أنها:
(أ) المهارات الإدراكية.           *(ب) المهارات الفنية.        (ج) المهارات الإنسانية.          (د) المهارات المالية والمحاسبية.

47. القدرة على إنجاز العمل مع الآخرين كأفراد وكمجموعات، والتواصل الفعال معهم، وتحفيزهم وقيادتهم، تندرج تحت تصنيف:
*(أ) المهارات الإنسانية.   (ب) المهارات الإدراكية (الإبداعية).     (ج) المهارات الفنية المتخصصة.   (د) السمات البيولوجية والفطرية.

48. القدرة على تخيل رؤية شاملة للمنشأة الناشئة، وتشخيص المشكلات وتحليل أسبابها، وابتكار أفكار استراتيجية تربط بين الأقسام، تمثل:
(أ) المهارة الفنية.       (ب) المهارة الإنسانية.         *(ج) المهارة الإدراكية (الإبداعية).       (د) المهارة الروتينية التشغيلية.

49. من السلوكيات الإسلامية التي يجب أن يراعيها رائد الأعمال في السوق امتثالاً للقاعدة الشرعية وحديث رسول الله ﷺ (لا ضرر ولا ضرار):
(أ) الاحتكار المتعمد لرفع الأسعار على المستهلكين.        *(ب) الالتزام بالأنظمة والقوانين وتجنب الإضرار بالناس أو المنافسين أو البيئة.
(ج) المبالغة في مدح السلعة بما ليس فيها لزيادة نسبة المبيعات.   (د) المماطلة في دفع أجور العاملين بحجة توفير السيولة النقدية للمشروع.

50. يقصد بـ "الكسب الحلال" ضمن أخلاقيات رائد الأعمال في الفقه الإسلامي:
(أ) بيع أي منتج أو خدمة تحقق ربحاً سريعاً بغض النظر عن تأثيرها الصحي أو المجتمعي.
*(ب) ابتعاد رائد الأعمال عن شبهات الحرام والغش والحرص على أداء الزكاة ودفع أجور العمال بلا مماطلة.
(ج) الاعتماد على القروض الربوية من البنوك التجارية لتوسيع المشروع بسرعة فائقة.
(د) تأخير أداء الديون والمستحقات للموردين لتعظيم السيولة النقدية للمنشأة.

51. التوجيه النبوي في قوله ﷺ: "إن الله يحب إذا عمل أحدكم عملاً أن يتقنه" يُعزز لدى رائد الأعمال قيمة مهنية تتمثل في:
(أ) الأمانة وتجنب كثرة الحلف في عمليات البيع والشراء.
*(ب) أداء العمل بدقة وإخلاص وتقديم منتجات مستوفية لمقاييس السلامة والجودة العالية.
(ج) حسن الخلق والاستقبال البشوش والابتسامة في وجه العميل.
(د) المسؤولية الاجتماعية عبر التبرع للأعمال الخيرية.

52. التاجر الذي يذكر محاسن المنتج بصدق، ويذكر كذلك قصوره أو عيوبه للعميل، ولا يخفي المعلومة أو يستغل جهل المشتري، يُطبق خلق:
(أ) المخاطرة العالية.    (ب) الابتكار الجذري في نماذج الأعمال.     *(ج) الأمانة.     (د) التخطيط المالي الاستباقي.

53. يُعرّف مصطلح "المسؤولية الاجتماعية" (Social Responsibility) في عالم ريادة الأعمال الحديث بأنه:
(أ) التركيز حصراً على تحقيق أقصى ربح ممكن للملاك دون أدنى اعتبار لمصالح العاملين.
*(ب) درجة الاهتمام بالناس والبيئة والمجتمع والمساهمة الملموسة في تنمية المجتمع المحلي وتجنب الإضرار به.
(ج) استخدام وسائل التواصل الاجتماعي لبناء حملات تسويقية وإعلانية فقط.
(د) التهرب القانوني من الالتزامات الضريبية والزكوية لزيادة التدفقات النقدية.

54. حرص رائد الأعمال على وضع عبارات توجيهية داخل محله مثل (ممنوع رمي النفايات) أو (اترك المكان نظيفاً) يعكس التزامه الجاد بـ:
(أ) مهارات التفكير الإدراكي والاستراتيجي.                            *(ب) آداب السلوك العام وحماية البيئة والنظافة العامة.
(ج) المهارات الفنية الخاصة بتصميم الديكورات الداخلية.                (د) البحث المستمر عن أفكار تسويقية جديدة.

55. يعتبر "الخوف من الفشل" عائقاً نفسياً كبيراً، ولذلك يتسم رائد الأعمال الناجح استراتيجياً بأنه:
(أ) يتخذ قرارات استثمارية عشوائية ومتهورة لإثبات شجاعته وكسر الخوف.
(ب) يتجنب إنشاء المشاريع تماماً مفضلاً البقاء في منطقة الراحة الوظيفية.
*(ج) ينظر للأمور بإيجابية وتفاؤل ويقبل المخاطرة المحسوبة التي تؤدي لنجاح كبير.
(د) يستثمر أمواله فقط في الودائع البنكية المضمونة أو السندات الحكومية الخالية من المخاطر.

56. إذا واجهت مشكلة إدارية معقدة داخل مشروعك، ووجدت في نفسك القدرة على تحليل أسبابها العميقة وإيجاد حل مبتكر يربط بين عدة أقسام بشكل متناغم، فأنت تعتمد على مهاراتك:
(أ) الفنية البحتة الدقيقة.      (ب) الإنسانية والاجتماعية.      *(ج) الإدراكية (المفاهيمية والتحليلية).   (د) القانونية والتشريعية.

57. رائد الأعمال الناجح يرفض الانسحاب عند أول عقبة تواجهه، ولا يستعجل حصد النتائج النهائية السريعة، هذا السلوك يترجم مباشرة سمة:
(أ) القدرة على الإقناع.    *(ب) المثابرة والإصرار.     (ج) المعرفة الفنية والإلمام بالتقنية.     (د) حسن الخلق وسعة الصدر.

58. ما هو التوازن الصحيح والمثالي لرائد الأعمال بشأن التعامل مع "وقت العمل" (كما أشارت إضاءة الوحدة المتعلقة بتجنب إضاعة الجهد)؟
(أ) تكريس كل الوقت المتاح للمشروع فقط، والتضحية الكاملة بالعائلة لتحقيق الثروة المنشودة.
*(ب) الاجتهاد في العمل مع إعطاء النفس وقتاً للراحة وعدم التضحية بالعائلة والحياة الاجتماعية لتجنب الانهيار.
(ج) العمل لمدة ساعتين يومياً فقط لتجنب الضغط النفسي وتوكيل جميع المهام لفريق العمل.
(د) الاعتماد على المرشدين والمستشارين بالكامل ليديروا المشروع نيابة عنه.

59. الشفافية المطلقة في الإعلان عن العيوب المحتملة في المنتج المُباع ترتبط ارتباطاً وثيقاً في الأخلاقيات المهنية الإسلامية بـ:
(أ) التدمير الذاتي المتعمد لسمعة المشروع التجاري.    
*(ب) خلق الأمانة وتجنب الغش اقتداءً بالهدي النبوي (من غش فليس مني).
(ج) وجود نقص وضعف واضح في مهارة الإقناع التسويقي لدى رائد الأعمال.
(د) الجهل بأساسيات ريادة الأعمال الحديثة التي تعتمد على تعظيم المبيعات بأي ثمن.

60. أي من السمات التالية تؤكد أن رائد الأعمال الناجح يجب أن يكون واسع الأفق، غير متعنت، ولديه قدرة مرنة على تغيير رأيه إذا ظهر له الأصوب أو الأفضل لشركته؟
(أ) التصلب والجمود الفكري في إثبات صحة الرأي الشخصي.    *(ب) تقبل النقد وتوسيع مدارك التفكير والاستفادة من المشورة.
(ج) تقبل المخاطر غير المحسوبة في الأسواق الجديدة.               (د) التمتع بالمهارات الفنية المتقدمة جداً في لغات البرمجة.
 
الوحدة الرابعة: الإبداع والابتكار

61. تُعرّف العملية الفكرية والذهنية البحتة التي تؤدي إلى إنتاج حلول أو أفكار أو طرق نظرية وعملية جديدة وفريدة بـ:
(أ) التنظيم الإداري.    *(ب) الإبداع (Creativity).     (ج) الاستثمار المالي.     (د) المحاسبة والتدقيق.

62. أي من العبارات التالية يُعد وصفاً صحيحاً لخصائص "الإبداع" كما ورد في المقرر؟
(أ) الإبداع موهبة فطرية تقتصر حصراً على الأشخاص عبقري الذكاء فقط.
(ب) الشخص يولد مبدعاً بالفطرة ولا يمكن تنمية أو تحفيز إبداعه بالتدريب.
(ج) يقتصر الإبداع على إنتاج الاختراعات التكنولوجية المعقدة فقط.
*(د) الإبداع موجود داخل كل فرد بدرجات مختلفة، ويمكن تنميته وصقله من خلال التدريب والتجريب.

63. القدرة على إنتاج عدد كبير وغزير جداً من الأفكار أو المترادفات في فترة زمنية محددة تُعرف بـ:
(أ) المرونة الفكرية.       (ب) الأصالة الإبداعية.      *(ج) الطلاقة (Fluency).      (د) مواصلة الاتجاه.

64. قدرة الفرد على التكيف السريع مع المواقف المستجدة، والتفكير في بدائل متنوعة، وتغيير طريقة استعمال الأشياء (وهي عكس حالة الجمود والتصلب الفكري) تُعرف بـ:
*(أ) المرونة (Flexibility).      (ب) الطلاقة التعبيرية.      (ج) الحساسية للمشكلات.      (د) الأصالة والجِدّة.

65. المبدع الذي يمتلك إحساساً مرهفاً يجعله يرى في الأشياء ما لا يراه الآخرون، وينظر للمواقف من زوايا متعددة ومبتكرة لاكتشاف الفجوات، يمتلك سمة:
(أ) مواصلة الاتجاه وتجاهل المشتتات.      *(ب) الحساسية للمشكلات.     (ج) الطلاقة اللفظية المعجمية.    (د) الجمود الفكري والروتين.

66. إنتاج شيء مبتكر أصيل وغير مقلد، والنفور التام من الحلول التقليدية المكررة للمشكلات، يُعبر بشكل دقيق عن سمة:
(أ) الطلاقة (التي تركز على كمية وغزارة الأفكار).               (ب) المرونة (التي تركز على التكيف).
*(ج) الأصالة (التي تركز على القيمة والنوعية والجدّة).          (د) الحساسية للمشكلات الاجتماعية.

67. القدرة العالية على التركيز لفترات طويلة في مجال الاهتمام رغم وجود المشتتات والمعوقات الخارجية التي تحبط العمل أحياناً تُعرف بـ:

الوحدة السادسة: تصميم نموذج الأعمال

101. يُعرّف مصطلح "نموذج الأعمال" (Business Model) في أدبيات ريادة الأعمال المعاصرة بأنه:
(أ) الهيكل التنظيمي الهرمي للموظفين داخل مقر الشركة الرئيس.
*(ب) الطريقة التي تستخدمها الشركة لإيجاد القيمة والربح والتميز عن المنافسين.
(ج) الحملة الإعلانية المؤقتة التي تطلقها الشركة عبر شبكات التواصل الاجتماعي.
(د) الميزانية العمومية الشاملة للشركة في نهاية كل سنة مالية.

102. أي من الخيارات التالية لا يُعد من الفوائد المباشرة لاستخدام مخطط (لوحة) نموذج الأعمال؟

(ب) توفير لغة مشتركة ونظرة شاملة ومبسطة بين أعضاء فريق العمل.
*(ج) تعقيد وإبطاء عملية الإعداد التي تحتاج عادة لشهور طويلة ومجلدات ضخمة.
(د) معرفة مدى ترابط المتغيرات والمجموعات وعلاقتها ببعضها داخل المشروع.

*(أ) المجموعات المختلفة من الناس أو المنشآت التي تستهدفها الشركة لتقدم لها المنتج أو الخدمة.
(ب) الموردين الخارجيين الذين يقدمون المواد الخام اللازمة للتصنيع.
(ج) القنوات التسويقية المتاحة للإعلان عن المنتجات.
(د) المبالغ المالية المحصلة من عمليات المبيعات اليومية.

(أ) الأنشطة الرئيسة التشغيلية.   *(ب) القيم المقترحة.  (ج) هيكل التكاليف المباشرة وغير المباشرة.  (د) الشراكات الرئيسة الاستراتيجية.

*(أ) القنوات.    (ب) مصادر الإيرادات والتدفقات النقدية.    (ج) الموارد الرئيسة الفكرية والمادية.    (د) الشراكات الرئيسة مع المنافسين.

106. ركن "العلاقات مع العملاء" (Customer Relationships) في مخطط نموذج الأعمال يركز على:

        function parseWordDataFormat() {
            const rawText = document.getElementById('raw-qa-data').textContent;
            // تقسيم على أساس "الوحدة" أو "أسئلة مراجعة للوحدة"
            let unitBlocks = rawText.split(/(?:أسئلة مراجعة للوحدة|الوحدة)\s*(الأولى|الثانية|الثالثة|الرابعة|الخامسة|السادسة|\d+)/i);
            if (unitBlocks.length < 2) unitBlocks = rawText.split(/\n(?=الوحدة|أسئلة مراجعة)/);
            const unitsData = [];
            let currentUnit = null;
            const optionRegex = /(\*?)\s*\(([أبجدتثجحخدذرزسشصضطظعغفقكلمنهوي1234567890]+)\)\s*(.*?)(?=\s*(?:\*?\s*\([أبجدتثجحخدذرزسشصضطظعغفقكلمنهوي1234567890]+\)|$))/gi;
            
            const lines = rawText.split('\n').map(l => l.trim()).filter(l => l !== '');
            for (let i = 0; i < lines.length; i++) {
                const line = lines[i];
                // بدء وحدة جديدة
                if (line.match(/^(الوحدة|أسئلة مراجعة للوحدة)/i)) {
                    if (currentUnit && currentUnit.questions && currentUnit.questions.length) unitsData.push(currentUnit);
                    let unitTitle = line.replace(/أسئلة مراجعة للوحدة|الوحدة/gi, '').trim();
                    if (!unitTitle) unitTitle = `الوحدة ${unitsData.length + 1}`;
                    currentUnit = { id: unitsData.length + 1, title: unitTitle, desc: `أسئلة تفاعلية لـ ${unitTitle}`, questions: [] };
                    continue;
                }
                if (!currentUnit) continue;
                // بدء سؤال جديد
                const qMatch = line.match(/^(\d+)\.\s+(.*)/);
                if (qMatch) {
                    const qId = parseInt(qMatch[1]);
                    let qText = qMatch[2];
                    const options = {};
                    let correct = null;
                    // البحث عن الخيارات في نفس السطر أو السطور التالية
                    let j = i+1;
                    let tempLines = [line];
                    while (j < lines.length && !lines[j].match(/^\d+\./) && !lines[j].match(/^(الوحدة|أسئلة مراجعة)/i)) {
                        tempLines.push(lines[j]);
                        j++;
                    }
                    const fullText = tempLines.join(' ');
                    let match;
                    optionRegex.lastIndex = 0;
                    while ((match = optionRegex.exec(fullText)) !== null) {
                        const isCorrect = match[1] === '*';
                        const optKey = match[2].trim();
                        const optText = match[3].trim();
                        options[optKey] = optText;
                        if (isCorrect) correct = optKey;
                    }
                    if (Object.keys(options).length > 0) {
                        currentUnit.questions.push({ id: qId, text: qText, options, correct });
                    } else {
                        // إذا لم يتم العثور على خيارات، نخزن السؤال بدون خيارات (لن يظهر)
                        currentUnit.questions.push({ id: qId, text: qText, options: {}, correct: null });
                    }
                    i = j-1;
                }
            }
            if (currentUnit && currentUnit.questions.length) unitsData.push(currentUnit);
            // تصفية الوحدات التي ليس بها أسئلة صالحة
            appState.units = unitsData.filter(u => u.questions.length > 0 && Object.keys(u.questions[0].options).length > 0);
        }

        function getOptionClass(qId, optionKey, correctKey) {
            const answer = appState.userAnswers[qId];
            if (!answer) return 'border-stone-200 hover:border-teal-400 hover:bg-teal-50 text-stone-700 bg-white';
            if (optionKey === correctKey) return 'border-emerald-500 bg-emerald-100 text-emerald-900';
            if (answer.selected === optionKey && optionKey !== correctKey) return 'border-rose-500 bg-rose-100 text-rose-900 line-through opacity-70';
            return 'border-stone-100 bg-stone-50 text-stone-400 opacity-50';
        }
        
        function handleAnswer(qId, selectedOption, unitId) {
            if (appState.userAnswers[qId]) return;
            let unit, question;
            for (let u of appState.units) {
                const found = u.questions.find(q => q.id === qId);
                if (found) { unit = u; question = found; break; }
            }
            if (!question) return;
            const isCorrect = (selectedOption === question.correct);
            appState.userAnswers[qId] = { selected: selectedOption, isCorrect: isCorrect };
            saveAnswersToStorage();
            updateSingleQuestionCard(qId, unitId);
            updateGlobalProgress();
            renderDashboard(); // تحديث المخططات
        }

            } else if (type === 'unit') {
                const unit = appState.units.find(u => u.id === id);
                if(unit) content += `<h2>${unit.title}</h2>` + buildQuestionsHTML(unit.questions); filename = unit.title;
            } else {

            }
            content += `</div>`;
            const blob = new Blob(['\ufeff' + content], { type: 'application/msword' });
            const link = document.createElement('a'); link.href = URL.createObjectURL(blob); link.download = filename + '.doc'; link.click(); URL.revokeObjectURL(link.href);
        }
